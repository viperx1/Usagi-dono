Features:
  • Hierarchical tree view: Anime titles as parents, episodes as children
  • Expand/collapse anime to show/hide episodes (▼ = expanded, ▶ = collapsed)
  • Sortable columns: Click column headers to sort
  • Alternating row colors for better readability
  • Real-time data: Shows current state from local database
  • Episode names: Displayed when available in database
  • State indicators: HDD, CD/DVD, Deleted, Unknown
  • Viewed status: Yes/No for each episode
  • Storage paths: Full file location for easy access

How it works:
  1. Click "Load MyList from Database" button
  2. Application queries local SQLite database
  3. Data is organized by anime → episodes
  4. Tree widget displays hierarchical structure
  5. No API calls needed for display (follows AniDB guidelines)
  6. Data was collected via FILE/MYLISTADD commands during hashing
